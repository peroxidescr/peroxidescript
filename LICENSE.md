## 🛠️ Troubleshooting Common Issues

- ❌ **Peroxide Script Gui Not Appearing After Execution**
  The overlay may be blocked by your executor's rendering engine. Make sure you are using an updated executor that supports drawing libraries. Restart Roblox and re-attach before loading the Peroxide Script again.

  **Solution:** Switch to a compatible executor and verify that the drawing API is enabled in its settings panel.

- ❌ **Auto Farm Stops Working Mid-Session**
  This usually happens when the Peroxide game server pushes a micro-update that changes mob spawn locations. The Peroxide Script farm route loses its path reference.

  **Solution:** Open the Peroxide Script Gui, navigate to Auto Farm settings, and click **Recalibrate Route** to refresh spawn data.

- ❌ **Peroxide Script For Mobile Crashes on Launch**
  Older Android versions with limited RAM may struggle to initialize all modules simultaneously. The Peroxide Script For Mobile needs at least 4 GB of available memory.

  **Solution:** Close background apps, clear your device cache, and relaunch the Peroxide Script application. Consider disabling ESP and Stat Tracker on low-end devices.

- ❌ **Kill Aura Targeting Friendly NPCs**
  The Peroxide Script combat filter may not recognize newly added NPCs from recent game updates until a patch is released.

  **Solution:** Enable `IgnoreTeammates = true` in the config and update to the latest Peroxide Script version to get the refreshed NPC whitelist.

- ❌ **Slow EXP Gain Despite Insta EXP Being Enabled**
  The EXP multiplier in the Peroxide Script Roblox tool caps at certain server-enforced limits. If the game detects unusual gain rates, it may throttle rewards temporarily.

  **Solution:** Lower the `Multiplier` value in the Peroxide Script config to `1.2` and use Auto Farm alongside it for steady, consistent progress.
